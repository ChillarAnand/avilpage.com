Tasks:

task.delay(arg, kwarg=value)
(A shortcut to .apply_async.)
t.apply_async((arg, ), {'kwarg': value})
(Run a task asynchronously) 
t.apply_async(countdown=10)
(Run asynchronously  10 seconds from now.)
t.apply_async(eta=now + timedelta(seconds=10))
#executes 10 seconds from now, specifed using eta
task.apply_async(countdown=60, expires=120)
#executes in one minute from now, but expires after 2 minutes.
task.apply_async(expires=now + timedelta(days=2))
#expires in 2 days, set using datetime.

Settings:

CELERY_ALWAYS_EAGER = True 
Tasks will be executed locally instead of being sent to the queue.

CELERY_IGNORE_RESULT = True
Not to store the task return values.
