There are hundreds of questions on StackOverflow about extracting text from html content but there is no single question on how to extract markup tags from html. After figuring out extracting using I thought of writing a post about it.

Here is a piece of code which does the job.
from bs4 import BeautifulSoup
from bs4.element import Tag

def get_markup(html):
    soup = BeautifulSoup(html)
    soup_gen = soup.recursiveChildGenerator()
    tags = [tag for tag in soup_gen if isinstance(tag, Tag)]
    return [str(t) if t.isSelfClosing else str(t).replace(t.string, '')
                    for t in tags]
Feed the html to BeautifulSoup, iterate over the soup elements to filter the tags. Once you got the tags, it is necessary to check if they are closed tags. If a tag is self closing, then it can taken out. On the other hand if it is not, the text between the tags has to be removed which leaves the tags behind.
